# Overview

## What the module does

**wp-module-coming-soon** provides a Coming Soon page for WordPress sites. It allows the site owner to enable or disable a “coming soon” state, typically showing a branded landing page to visitors while the site is being built. The module registers with the Newfold Module Loader and is used by brand plugins (e.g. Bluehost); the host can customize template args via the `newfold/coming-soon/filter/args` filter.
